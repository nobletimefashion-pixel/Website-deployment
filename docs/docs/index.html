Features

Everything you need, right where you work

Nexus Agent lives in your terminal and works with your existing tools. No context switching, no complex setup.

File Operations

Read, write, edit files with diff-based precision. Glob, grep, and list directory tools for fast codebase exploration.

Shell Execution

Run commands, tests, and builds. Dangerous command detection with configurable approval policies.

Web & Network

Web search, fetch, scrape, and browser automation. Gather information programmatically from any URL.

OSINT Suite

Domain, email, IP, phone, username, WHOIS, Censys, Shodan, and social media intelligence tools.

MCP Integration

Connect external MCP servers via stdio or SSE. Extend the agent with custom tools dynamically.

RAG — PDF Querying

Ingest PDFs and query them with vector search. Ask natural language questions about your documents.

Sub-Agents

Spawn focused child agents for investigation and code review. Each runs with isolated context.

Hook System

Trigger scripts at 5 lifecycle events. Context passed via environment variables for unlimited extensibility.

Dual UI — TUI + Web

Rich terminal TUI with theming, plus a FastAPI + WebSocket web UI. Same agent, two interfaces.
